### 📊 Observations

From the results obtained:

* **Hashing** was the fastest search method
* **Binary Search** was the second fastest
* **Linear Search** was the slowest

This aligns with theoretical expectations:

* Hashing → *O(1)* average time complexity
* Binary Search → *O(log n)*
* Linear Search → *O(n)*

---

### ✅ Conclusion

Hashing proved to be the most efficient searching technique in this experiment due to its constant-time average performance. Binary search also performed efficiently but requires sorted data. Linear search, while simple, is less efficient for large datasets.
